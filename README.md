## 프로젝트 소개

"GhostWriter"는 교수님께 이메일을 보내는데 어려움을 겪는 대학생을 위한 **문구 추천 웹 서비스**입니다.  
메일을 쓸 때, 상황별로 적절한 문구를 추천해주는 것뿐만 아니라, 기본적으로 어떻게 메일을 써야하는지도 소개하고 있습니다.
